# Code Package for CAGE: An Empirical Study of the Conversation-Action Gap in Agentic LLMs

## Description
This directory contains the experiment project used for the paper.

## Project Files
- `baselines.py`
- `data_loader.py`
- `experiment_harness.py`
- `main.py`
- `methods.py`
- `metrics.py`
- `models.py`

## How to Run
`python main.py`

## Dependencies
Install dependencies with `pip install -r requirements.txt` if needed.
